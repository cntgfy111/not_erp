## Quick Start

Requirments:
- bash
- Python 3.10
- Docker *какой-то*
- docker-compose 1.29.2

To start service just clone repositiry and run `run.sh` script:
```bash 
git clone git@github.com:cntgfy111/not_erp.git
cd not_erp && ./run.sh
```
